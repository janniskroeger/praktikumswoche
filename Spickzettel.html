1. Das Dokument

Ein HTML-Dokument beginnt immer mit der <!DOCTYPE html> Deklaration, gefolgt von dem <html> Tag:

<!DOCTYPE html>
<html>
  ...
</html>

2. Kopf des Dokuments

Der Kopf (Head) des Dokuments enthält Metadaten, wie Titel, Zeichensatz und Stylesheets:

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Dokumenttitel</title>
  <style>
    /* CSS goes here */
  </style>
</head>

3. Überschriften

Überschriften werden mit den Tags <h1> bis <h6> definiert:

<h1>Größte Überschrift</h1>
<h2>Zweitgrößte Überschrift</h2>
...
<h6>Kleinste Überschrift</h6>

4. Absätze

Absätze werden mit dem <p> Tag erstellt:

<p>Das ist ein Absatz.</p>

5. Links

Links werden mit dem <a> Tag erstellt:

<a href="https://www.example.com">Besuchen Sie Beispiel.com</a>

6. Bilder

Bilder werden mit dem <img> Tag eingefügt:

<img src="bild.jpg" alt="Beschreibung des Bildes">

7. Listen

Es gibt ungeordnete Listen <ul> und geordnete Listen <ol>:

<ul>
  <li>Listenelement 1</li>
  <li>Listenelement 2</li>
</ul>

<ol>
  <li>Erstes Element</li>
  <li>Zweites Element</li>
</ol>

8. Tabellen

Tabellen werden mit dem <table> Tag erstellt:

<table>
  <tr>
    <th>Überschrift 1</th>
    <th>Überschrift 2</th>
  </tr>
  <tr>
    <td>Daten 1</td>
    <td>Daten 2</td>
  </tr>
</table>

9. Textformatierung

Wichtige Textformatierungselemente sind <b>, <strong>, <i>, und <div>:

<b>Fetter Text</b> - Dieser Text ist fett.
<strong>Wichtiger fetter Text</strong> - Dieser Text ist ebenfalls fett und hat semantisch mehr Bedeutung.
<i>Kursiver Text</i> - Dieser Text ist kursiv.
<div>Blockelement</div> - Dieses Element ist ein Container für andere Elemente.

10. Div-Element

Das <div> Element ist ein allgemeines Block-Container-Element:

<div>Dieses Element kann als Container für andere Elemente verwendet werden.</div>

11. Inline Styles

Stile können direkt auf Elemente angewendet werden, indem das style Attribut verwendet wird:

<p style="color: red;">Dieser Text ist rot.</p>
<p style="font-size: 20px;">Dieser Text hat eine Schriftgröße von 20px.</p>
<p style="background-color: yellow;">Dieser Text hat einen gelben Hintergrund.</p>
<p style="margin: 20px;">Dieses Div hat einen Außenabstand von 20px.</p>
<p style="padding: 10px;">Dieses Div hat einen Innenabstand von 10px.</p>
<p style="border: 1px solid black;">Dieses Div hat einen schwarzen Rand.</p>
<div style="width: 100px; height: 100px; background-color: blue;">Dieses Div ist 100x100 Pixel groß und blau.</div>

12. CSS-Klassen

CSS-Klassen werden verwendet, um mehrere Elemente mit denselben Stilen zu versehen. Hier sind einige Beispiele für CSS-Klassen:

<div class="rot">Dieser Text ist rot.</div>
<div class="großer-text">Dieser Text hat eine Schriftgröße von 20px.</div>
<div class="gelber-hintergrund">Dieser Text hat einen gelben Hintergrund.</div>
<div class="außenabstand">Dieses Div hat einen Außenabstand von 20px.</div>
<div class="innenabstand">Dieses Div hat einen Innenabstand von 10px.</div>
<div class="rand">Dieses Div hat einen schwarzen Rand.</div>
<div class="blaues-div">Dieses Div ist 100x100 Pixel groß und blau.</div>

13. Flexbox Layout

Flexbox-Layouts erleichtern die Erstellung von flexiblen und anpassbaren Layouts. Hier sind einige Beispiele:

<div class="row">
    <div class="col-6 ">.col-6</div>
    <div class="col-6 ">.col-6</div>
</div>
<div class="row">
    <div class="col-4">.col-4</div>

    <div class="col-4">.col-4</div>
</div>
<div class="row">

    <div class="col-3">.col-3</div>
    <div class="col-3">.col-3</div>
    <div class="col-3">.col-3</div>
</div>
